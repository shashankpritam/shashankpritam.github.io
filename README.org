#+TITLE: shashankpritam.github.io

Personal website for [[https://shashankpritam.github.io][Shashank Pritam]].

Built with Emacs [[https://orgmode.org/manual/Publishing.html][org-publish]].

* Structure

#+BEGIN_EXAMPLE
org/           ← source pages (.org)
static/
  css/         ← Stylesheet
  pics/        ← images
publish.el     ← org-publish build script
public/        ← generated HTML
#+END_EXAMPLE

* Build locally

Requires Emacs (29+) and [[https://d2lang.com][D2]].

#+BEGIN_SRC sh
# install D2 once
curl -fsSL https://d2lang.com/install.sh | sh -s -- --version v0.7.1

emacs --batch --load publish.el
# output in public/
#+END_SRC

Open =public/index.html= in a browser to preview.

* Deploy

Pushing to =main= triggers the GitHub Actions workflow (=.github/workflows/pages.yml=), which installs =emacs-nox= and runs =publish.el=. Output is deployed to GitHub Pages automatically.

* Theme

The site uses [[https://github.com/wintermute-cell/magick.css][magick.css]] by winterveil as its structural base, with a custom warm color scheme layered on top.

** Color palette

|--------------+-----------+----------------------------|
| Background   | =#ECE7CD= | warm ivory                 |
| Text         | =#4a4540= | warm brownish gray         |
| Links        | =#7c6f4e= | warm olive                 |
| Visited      | =#5a4e3e= | dark olive-brown           |

Originally based on the [[https://github.com/oxalorg/sakura][Sakura CSS]] palette, then shifted uniformly to warmer, slatier tones for an easy-on-the-eyes feel.

** Dark mode

A =prefers-color-scheme: dark= block overrides the palette variables with a warm, dimmed variant, so readers whose OS is in dark mode get a matching theme automatically. It is pure CSS — no toggle, no JavaScript. D2 diagrams are lightened with a CSS filter so their build-time ink stays legible; photographs are left untouched.

* No JavaScript

The site ships no executable JavaScript. Everything interactive-looking — framed, scrollable pictures, decorated section subtitles, math, and diagrams — is plain HTML and CSS rendered at build time. The only =<script>= element is a =type="application/ld+json"= structured-data block (inert metadata for search engines).

A =Content-Security-Policy= meta tag enforces this: =script-src 'none'=, everything same-origin, with Google Fonts the sole third-party origin allowed for styles and fonts.

* Math and diagrams

Math and diagrams are rendered at build time via [[https://d2lang.com][D2]] — no JavaScript required.
D2 is installed by the CI workflow (=v0.7.1= to =~/.local/bin=) and must also be on =PATH= for local builds.

** Inline math

Use a =d2math= source block. Write a bare LaTeX expression (no =$= delimiters):

#+BEGIN_SRC org
,#+BEGIN_SRC d2math
p^2 + 2pq + q^2 = 1
,#+END_SRC
#+END_SRC

The block renders as a compact inline SVG, sized relative to the surrounding text.

** Diagrams

Use a =d2= source block with standard [[https://d2lang.com/tour/intro][D2 syntax]]:

#+BEGIN_SRC org
,#+BEGIN_SRC d2
TE invasion: {
  shape: sequence_diagram
  Host -> piRNA cluster: insertion bias
  piRNA cluster -> TE: silencing
  TE -> Host: fitness cost
}
,#+END_SRC
#+END_SRC

The block renders as a block-level SVG. Diagram backgrounds are transparent and blend with the page theme.
