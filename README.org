#+TITLE: shashankpritam.github.io

Personal website for [[https://shashankpritam.github.io][Shashank Pritam]].

Built with Emacs [[https://orgmode.org/manual/Publishing.html][org-publish]].

* Build locally

Requires Emacs (29+) and [[https://d2lang.com][D2]].

#+BEGIN_SRC sh
# install D2 once
curl -fsSL https://d2lang.com/install.sh | sh -s -- --version v0.7.1

emacs --batch --load publish.el
# output in public/
#+END_SRC

Open =public/index.html= in a browser to preview.

* Deploy

Pushing to =main= triggers the GitHub Actions workflow (=.github/workflows/pages.yml=), which installs =emacs-nox= and runs =publish.el=. Output is deployed to GitHub Pages automatically.

* Theme

The site uses [[https://github.com/wintermute-cell/magick.css][magick.css]] by winterveil as its structural base, with a custom warm color scheme layered on top.

* Math and diagrams

Math and diagrams are rendered at build time via [[https://d2lang.com][D2]] — no JavaScript required.
D2 is installed by the CI workflow (=v0.7.1= to =~/.local/bin=) and must also be on =PATH= for local builds.

** Inline math

Use a =d2math= source block. Write a bare LaTeX expression (no =$= delimiters):

#+BEGIN_SRC org
,#+BEGIN_SRC d2math
p^2 + 2pq + q^2 = 1
,#+END_SRC
#+END_SRC

The block renders as a compact inline SVG, sized relative to the surrounding text.

** Diagrams

Use a =d2= source block with standard [[https://d2lang.com/tour/intro][D2 syntax]]:

#+BEGIN_SRC org
,#+BEGIN_SRC d2
TE invasion: {
  shape: sequence_diagram
  Host -> piRNA cluster: insertion bias
  piRNA cluster -> TE: silencing
  TE -> Host: fitness cost
}
,#+END_SRC
#+END_SRC

The block renders as a block-level SVG. Diagram backgrounds are transparent and blend with the page theme.
